# 个人生活看板 PWA

这是个人生活看板的公开程序外壳，不包含任何个人任务、习惯、项目或就医信息。

## 使用方式

1. 打开 GitHub Pages 地址。
2. 使用“导入数据”选择由本地看板生成的 JSON。
3. 在手机浏览器菜单中选择“添加到主屏幕”。
4. 后续导入新的内容更新包时，已有任务完成状态和习惯历史会保留；导入完整备份时则恢复其中的状态。

个人数据保存在手机浏览器的 `localStorage` 中。更换手机或清理浏览器数据前，请使用“导出数据”保存完整备份。

## 隐私

本仓库只包含网页程序和空数据结构。请勿提交个人数据 JSON、导出备份或包含真实生活信息的 `dashboard-data.js`。
